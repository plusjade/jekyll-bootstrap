The quickest and most hassle-free way to get your new Jekyll powered website up and running. 100% compatible with GitHub pages. (No idea what Jekyll is? read the intro)

Why Use Jekyll-bootstrap?

Experienced Jekyll users recommend cloning an existing blog in order to get up and running the fastest. Jekyll-bootstrap takes this advice to the next level by attempting to be the definitive Jekyll-framework to clone which comes complete with guides and code examples for the most common tasks.

Core Benefits

  • Convenient
    Jekyll-bootstrap comes with code examples for all of the most common tasks you'd want in a basic blogging engine ready to customize - it's plug and play.
  • Comprehensive
    Jekyll-bootstrap explains how and why jekyll works the way it does, empowering you to extend as you fit without the barrier of a high learning curve.
  • Easy Deploy to GitHub
    Jekyll-bootstrap is 100% compatible with deploying to GitHub. Just push your repository to a valid GitHub Pages endpoint and GitHub hosts your website <3.

Start Now!

1 - Install

  $ gem install jekyll
  $ git clone https://plusjade@github.com/plusjade/jekyll-bootstrap.git
  $ cd jekyll-bootstrap
  $ jekyll --server

Need installation help? Check out the installation documentation and/or contact me through github.

2 - Run on Localhost

Access your localhost version of Jekyll-bootstrap at http://localhost:4000/

3 - Explore and Customize

Read the pages, try the examples, check the source code, then clone and customize.
